FAQ

Preguntas frecuentes

¿Qué puedes contratar con Onboarding Digital?

• Extracción de datos por medio de reconocimiento de caracteres (OCR): Comprobantes de domicilio, identificaciones, ingresos, entre otros.
• Validación: Biométrica, dactilar o facial.
• Validación de identificaciones ante organismos oficiales locales.
• Firma electrónica y generación de contrato.

¿Qué puedes contratar con Compliance?

• Consulta una o múltiples listas negras.
• Consulta de OFAC, PEP, GAFI; Más de 40 listas.
• Consultas de listas nacionales e internacionales.
• Cumple con las disposiciones oficiales de los organismos reguladores.

¿Qué puedes contratar con Credit Risk?

• Conexión a uno o múltiples bureaus de crédito.
• Creación de árboles de decisión para determinar la aceptación o rechazo de una solicitud de crédito.
• Capacidad para integrar modelos de riesgo con diferentes tipos de fuentes (Listas, bureaus) en bases de datos externas o internas.
• Uso de modelos ya generados en: Python, Programación R, Regresión logística, Machine Learning, etc.
